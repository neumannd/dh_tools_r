nDaysOfMonthR Documentation

Returns the number of days per month

Description

Returns the number of days per month.

Usage

nDaysOfMonth(month, year = 2001)

Arguments

month

numeric or array of numerics between 1 and 12

year

numeric, optional [default: 2001; no leap year]

Value

numeric or array of numerics giving the number of days per month

Examples


# #days in may 2001:
nDaysOfMonth(5)
> 31

# #days in February 2001:
nDaysOfMonth(2)
> 28

# #days in February 2004:
nDaysOfMonth(2, 2004)
> 29

# #days in a set of months in 2007
nDaysOfMonth(c(1,2,6,12), 2007)
> 31 28 30 31
