mnbR Documentation

Mean Normalized Bias

Description

Calculate a mean normalized bias.

Usage

mnb(p, o, na.rm = TRUE)

Arguments

p

numeric array; predicted values (model)

o

numeric array; measured/observed values

na.rm

a logical indicating whether missing values should be removed.

Details

MNB = 1/n * sum((p-o)/o)

Value

numeric; mean normalized bias

Examples


  # ex 1
  p <- 1:5
  o <- 2:6
  mnb(p,o)
  > -0.29
  
  # ex 2
  p <- c(1:4, NA)
  o <- 2:6
  mnb(p,o)
  > -0.3208333
  
  # ex 3
  p <- c(1:4,NA)
  o <- 2:6
  mnb(p,o,na.rm=FALSE)
  > NA
  
  # ex 4
  p <- 1:5
  o <- c(2:5, 0)
  mnb(p,o)
  > -0.3208333
  
  # ex 5
  p <- 1:5
  o <- c(2:5, 0)
  mnb(p,o, na.rm=FALSE)
  > Inf
