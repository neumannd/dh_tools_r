rowMinsR Documentation

Form Row Minima

Description

Calculates the minimum of each row of a matrix, a data.frame, or any other 2-dimensional structure.

Usage

rowMins(x, na.rm = FALSE)

Arguments

x

data.frame or array of two dimensions, containing numeric, complex, integer or logical values.

na.rm

logical: a logical indicating whether missing values should be removed.

Value

A numeric or complex data.frame/vector. For min and max if there are only numeric inputs and all are empty (after possible removal of NAs), the result is double (Inf or -Inf), consistent with min and max.

Author(s)

Daniel Neumann, daniel.neumann@io-warnemuende.de

See Also

colMaxs, rowMaxs, colMins, colMeans, rowMeans, colSums, rowSums

Examples

 abc = array(1:30, dim = c(6,5))
 rowMins(abc)
 # returns:  1  2  3  4  5  6
 
 def = as.data.frame(abc)
 rowMins(def)
 # returns
 #    1  2  3  4  5  6
