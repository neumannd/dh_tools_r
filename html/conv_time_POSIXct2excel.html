conv_time_POSIXct2excelR Documentation

Convert POSIXct Format to MS Excel time stamp

Description

Convert POSIXct Format to MS Excel time stamp

Usage

conv_time_POSIXct2excel(time,
  reftime = as.POSIXct("1900-01-01 00:00 UTC"))

Arguments

time

POSIXct; time in format POSIXct

reftime

POSIXct, optional; provide another reference time

Value

time numeric; time as MS Excel time stamp in hours since 01/01/1900

Examples


  # 2001-03-29 23:00:00 UTC => 36980 hours since 1900-01-01 00:00 UTC
  time_posixct <- as.POSIXct('2001-03-29 23:00:00 UTC', tz='UTC')
  time_excel <- conv_time_POSIXct2excel(time_posixct)
  print(time_excel)
