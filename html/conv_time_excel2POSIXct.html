conv_time_excel2POSIXctR Documentation

Convert MS Excel time stamp to POSIXct Format

Description

Convert MS Excel time stamp to POSIXct Format.

Usage

conv_time_excel2POSIXct(time,
  reftime = as.POSIXct("1900-01-01 00:00 UTC"))

Arguments

time

numeric; a time stamp in hours since 01/01/1900

reftime

POSIXct, optional; provide another reference time

Value

time_POSIXct POSIXct: time in format POSIXct

Examples


  # 31069 hours since 01/01/1900 00:00:00 => 22/01/1985 
  time_excel <- 31069 
  time_posixct <- conv_time_excel2POSIXct(time_excel)
  print(time_posixct)
